# 🏗️ ArciSpace Platform

A modern **Home Improvement & E-Commerce Platform** built with **React (Vite + TypeScript)** that connects users with products, services, engineers, and workers for home renovation and construction.

---

## 🚀 Overview

ArciSpace is a complete platform that allows users to:

- 🛒 Browse and purchase home improvement products  
- 👷 Connect with engineers, workers, and service providers  
- 🏠 Request renovation services (kitchen, furniture, design)  
- 🤖 Use AI-powered pages for assistance  
- 🌍 Switch between Arabic & English with full RTL support  

---

## ✨ Features

### 🔐 Authentication System
- Login / Register  
- Forgot Password / OTP Verification  
- Secure authentication with token  

---

### 🛒 E-Commerce System
- View products (furniture, tools, paint, etc.)  
- Add to cart & manage items  
- Product details with colors, dimensions, and stock  
- Wishlist support  

---

### 🔍 Filtering & Search
- Filter by category, price, and color  
- Search for products easily

### 👷 Services & Professionals

- Request services directly

### 🏠 Renovation Services

### 👤 User Profile
- Edit profile information

### 🤖 AI Features

- Smart suggestions for home improvement

### 🌍 Multi-Language Support

## 🛠️ Tech Stack

- ⚛️ React 18 + Vite  
- 🟦 TypeScript  
- 🎨 Tailwind CSS + shadcn/ui + Radix UI  
- 🌐 Axios / Fetch API  
- 🔄 React Router  
- 📊 TanStack Query  
- 🧾 React Hook Form + Yup / Zod  
- 🎞️ Framer Motion  
- 🌍 i18next (Multi-language)  

---

- Authentication APIs  
- Products APIs (Filter / Search / CRUD)  
- Services APIs (Engineers / Workers)
